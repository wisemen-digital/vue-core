# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-03-13

### Added

- Initial release
- `createLocaleFactory` for standardized locale configuration management
- `DeepPartial` utility type for recursive partial objects
- `LocaleConfig` type for locale configuration structures
- Support for any locale codes (flexible, not limited to predefined locales)
- Built-in validation for locale configurations
- Full TypeScript support with generic types
- Comprehensive documentation and usage patterns
- Examples for Zod, vue-i18n, and custom message systems
